Git Dashboard

自动发现本地所有 Git 仓库，以可视化面板独立展示每个项目的每日提交量

Go React SQLite
下载应用 GitHub

功能特性

🔍

自动发现仓库

无需手动添加项目。设置扫描目录后自动递归发现所有 Git 仓库，支持 Windows/macOS/Linux 跨平台默认规则。

📊

可视化面板

每个项目独立卡片展示提交量（新增/删除/净增行数）。趋势折线图追踪每日代码产出变化。

📦

智能项目分组

自动识别 Monorepo 与单仓库项目。支持手动调整项目目录级别，灵活控制展示粒度。

📅

工作日检查

自动区分工作日与周末。设定每日代码量标准，未达标时面板给出提醒。

💻

跨平台单文件

Go 编译为单个二进制文件，无需运行时依赖。双击启动，自动打开浏览器访问面板。

📦

PWA 可安装

支持 PWA，启动后可安装到桌面或移动设备主屏幕，获得原生应用般的使用体验。

命令行快速安装

macOS / Linux

curl -fsSL https://raw.githubusercontent.com/sky-jiangcheng/gitboard/master/scripts/install.sh | bash

Windows (PowerShell)

iwr -useb https://raw.githubusercontent.com/sky-jiangcheng/gitboard/master/scripts/install.ps1 | iex

手动安装

下载对应平台的二进制文件，赋予执行权限后直接运行即可。

chmod +x gitboard && ./gitboard

使用说明

1. 启动应用

下载后双击运行，应用会自动打开浏览器访问本地面板（默认端口 18731）。

2. 配置扫描目录

首次启动使用平台默认规则。前往设置页面添加或修改扫描根目录。

3. 查看统计

仪表盘展示所有项目卡片。点击卡片进入详情页查看趋势图和子仓库明细。
