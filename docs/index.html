GitBoard

自动发现本地 Git 仓库，可视化每日提交量；内置知识库、仓库知识挖掘与命令面板，把每个项目变成可检索的知识。

v1.5.0 Go React SQLite
下载应用 GitHub

功能特性

🔍

自动发现仓库

设置扫描根目录后递归发现所有 Git 仓库，平台自适应默认规则；智能分组识别 Monorepo 与单仓库，支持事务安全的拆分/合并。

🎯

每日目标进度

仪表盘首屏以进度环突出「今天达标了吗」。项目卡片展示新增/删除/净增、目标进度与达标状态，工作日未达标即时提醒。

📚

知识库

跨项目笔记中心：Markdown 笔记、标签、置顶、分类。全文搜索带排序与片段高亮，覆盖笔记与待办，⌘/Ctrl+K 命令面板秒达。

⛏️

仓库知识挖掘

自动提取 README、检测技术栈与语言占比、最近提交流，结果缓存避免重复扫描，把仓库本身变成可浏览的知识。

📅

提交热力图与趋势

GitHub 风格的 52 周提交热力图，点击某天即筛选当日项目；趋势折线图追踪每日代码产出变化。

💻

跨平台单文件

Wails + Go 编译为单个二进制，无运行时依赖；PWA 可安装到桌面/主屏幕。一键导入 Claude 记忆为知识笔记。

安装说明

macOS

  1. 从 Releases 下载 gitboard-darwin-amd64.dmg（Intel）或 darwin-arm64.dmg（M 系列）
  2. 双击 .dmg 打开，将 GitBoard.app 拖入 Applications
  3. 首次打开：右键点击 > 打开（绕过 Gatekeeper 未签名提示）

Windows

  1. 从 Releases 下载 gitboard-windows-amd64.zip
  2. 解压后双击 gitboard.exe 运行
  3. Windows Defender 弹出警告时，点击 更多信息 > 仍要运行

Linux

下载 .tar.gz 后：

tar -xzf gitboard-linux-amd64.tar.gz && chmod +x gitboard && ./gitboard

系统安装（可选）：

sudo mv gitboard /usr/local/bin/

一键安装（实验性）

macOS / Linux：

curl -fsSL https://raw.githubusercontent.com/sky-jiangcheng/gitboard/master/scripts/install.sh | bash

Windows (PowerShell 管理员)：

iwr -useb https://raw.githubusercontent.com/sky-jiangcheng/gitboard/master/scripts/install.ps1 | iex

使用说明

1. 启动应用

下载后双击运行，GitBoard 以原生窗口打开，自动扫描并展示所有 Git 仓库。

2. 配置与达标

设置扫描目录、每日代码量标准与 Git 作者。仪表盘首屏进度环即时反映今日目标达成情况。

3. 挖掘项目知识

进入项目详情查看自动提取的 README、技术栈、语言占比与最近提交；侧栏管理待办与笔记。

4. 知识库与搜索

「知识库」页跨项目浏览/筛选笔记；⌘/Ctrl+K 呼出命令面板，搜索笔记与待办并跳转项目。
